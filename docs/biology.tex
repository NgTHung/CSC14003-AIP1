\section{Biology-Based Algorithms}

Biology-based algorithms draw inspiration from collective behaviours observed in nature—swarm intelligence, foraging strategies, and pheromone-mediated communication. This subsection presents five algorithm families implemented in the project: Particle Swarm Optimisation (PSO), Artificial Bee Colony (ABC), Cuckoo Search (CS), Firefly Algorithm (FA), and Ant Colony Optimisation (ACO) with three variants.
